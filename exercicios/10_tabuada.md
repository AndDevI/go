# Exercicio 10 - Tabuada

**Objetivo:** Ler um inteiro `n` e imprimir a tabuada de 1 a 10.

**Entrada:** Um inteiro `n`.

**Saida:** 10 linhas no formato `n x i = resultado`.

**Exemplo**
Entrada:
```
3
```
Saida:
```
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
3 x 4 = 12
3 x 5 = 15
3 x 6 = 18
3 x 7 = 21
3 x 8 = 24
3 x 9 = 27
3 x 10 = 30
```
